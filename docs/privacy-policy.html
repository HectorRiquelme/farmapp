Política de Privacidad

Última actualización: 23 de marzo de 2026

FarmApp es una aplicación móvil gratuita desarrollada por Héctor Riquelme. Esta política describe cómo la aplicación maneja la información del usuario.

1. Datos que recopila la aplicación

FarmApp solicita acceso a la ubicación del dispositivo exclusivamente mientras se usa la aplicación, con el único propósito de calcular la distancia hacia las farmacias de turno más cercanas.

  • La ubicación se procesa localmente en el dispositivo.
  • La ubicación no se almacena en ningún servidor ni se transmite a terceros.
  • La ubicación no se guarda en la base de datos local de la aplicación.

2. Datos que NO recopila la aplicación

  • No se recopila información personal (nombre, correo, teléfono del usuario).
  • No se requiere crear cuenta ni iniciar sesión.
  • No se recopilan datos de uso, analytics ni telemetría.
  • No se utilizan cookies.
  • No se muestra publicidad.
  • No se realizan compras dentro de la aplicación.

3. Almacenamiento local

La aplicación almacena datos localmente en el dispositivo mediante una base de datos SQLite:

  • Farmacias de turno: información pública obtenida del Ministerio de Salud de Chile (MINSAL), guardada temporalmente para permitir el uso sin conexión. Estos datos se eliminan automáticamente después de 2 días.
  • Coordenadas geocodificadas: direcciones convertidas a coordenadas geográficas, almacenadas como caché para evitar consultas repetidas.

Estos datos se almacenan únicamente en el dispositivo del usuario y no se transmiten a ningún servidor.

4. Servicios de terceros

FarmApp se conecta a los siguientes servicios externos:

  • API MIDAS del Ministerio de Salud de Chile (MINSAL): para obtener el listado público de farmacias de turno. No se envían datos personales en esta consulta.
  • Nominatim (OpenStreetMap): para convertir direcciones de farmacias en coordenadas geográficas. La consulta incluye solo la dirección de la farmacia, nunca datos del usuario. La dirección IP del dispositivo es visible para el servidor como parte de cualquier conexión HTTP.
  • CartoDB (tiles de mapa): para mostrar el mapa visual. No se envían datos personales.

5. Permisos del dispositivo

  • Ubicación (GPS): para calcular la distancia a las farmacias. Se solicita permiso explícito al usuario antes de acceder.
  • Internet: para consultar la API de farmacias de turno y los tiles del mapa.
  • Teléfono: para permitir llamar directamente a una farmacia desde la aplicación, solo cuando el usuario lo solicita.

6. Seguridad

Al no existir servidores propios ni recopilación de datos personales, los riesgos de seguridad son mínimos. Toda la información se procesa y almacena localmente en el dispositivo del usuario.

7. Privacidad de menores

La aplicación no está dirigida a menores de 13 años y no recopila información de ningún usuario, independientemente de su edad.

8. Cambios en esta política

Esta política puede actualizarse ocasionalmente. Los cambios se publicarán en esta misma página con la fecha de actualización correspondiente.
